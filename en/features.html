Features

The current features of PaintField are:

  • Tabs
  • Navigator (Canvas rotation & scaling)
  • Layers with groups
  • Infinite undo / redo
  • Layer move
  • Simple brush, which can be used with Wacom tablets
  • ...Too few features?
  • Please wait until more features are added...
