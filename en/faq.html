Frequently Asked Questions

What are the default shortcuts of PaintField?

  • Translating canvas: Space+Drag
  • Scaling canvas: Option+Drag (Mac), Control+Drag (Linux)
  • Rotating canvas: Shift+Drag

Also, many of the menu bar items has their own shortcuts.

PaintField fails to launch. What can I do?

On Linux:

The following libraries must be installed. Check if these libraries are installed and install them if not.

  • Qt 4.8
  • libfreeimage
  • libqjson

On Mac:

PaintField only supports Lion and Mountain Lion.

How can I do to change the settings?

PaintField searches 2 folders on startup to load settings or brush presets in this order:

  1. The folder that contains PaintField executable (PaintField.app/Contents/MacOS on Mac) (built-in)
  2. The folder named "PaintField" in user's document folder (~/Documents/PaintField on Mac) (for user)

The structure of each folders is like this:

Contents/
  Brush Presets/
      (brush preset files and folders)
Settings/
  key-bindings.json (key binding settings)
  menubar.json (menubar arrangement)
  workspace-items.json (toolbars and sidebars arrangement)

Place your original settings files in the user PaintField folder, and the settings are overridden.
