PaintField

An open-source cross-platform digital painting / image editing app

Simple UI

The user interface of PaintField is very simplified and easy-to-use.

Free and Free

PaintField is free of charge and open-source. The source code is licensed under LGPL.

Cross-platform

Currently PaintField supports both Mac and Linux, and support for Windows is planned.

Features

PaintField is designed for enjoying paiting.

  • Tabs
  • Navigator (Canvas rotation & scaling)
  • Layers with groups
  • Infinite undo / redo
  • Layer move
  • Simple brush, which can be used with Wacom tablets

Usage

Default Keys

  • Space+Drag: Move canvas
  • Alt(Option)+Space+Drag: Scale canvas
  • Shift+Space+Drag: Rotate canvas

How to change settings or add brush presets

PaintField searches 2 folders on startup to load settings or brush presets in this order:

  1. The folder that contains PaintField executable (PaintField.app/Contents/MacOS on Mac) (built-in)
  2. The folder named "PaintField" in user's document folder (~/Documents/PaintField on Mac) (for user)

The structure of each folders is:

Contents/
  Brush Presets/
    (brush preset files and folders)
Settings/
  key-bindings.json (key binding settings)
  menubar.json (menubar arrangement)
  workspace-items.json (toolbars and sidebars arrangement)

Setting files placed in the user folder overrides the built-in settings.
